# TechFlag IDP Community v20260621

This release prepares the public community edition for GitHub distribution.

## Highlights

- Local-first intelligent document processing workbench.
- Default community setup with SQLite and local object storage.
- Configurable MinerU document parsing provider.
- Bring-your-own OpenAI-compatible LLM for AI extraction.
- Evidence-grounded extraction result display.
- Chinese and English frontend UI.
- Public README, user manuals, and safe product screenshots.

## Startup

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -r requirements.txt
cp .env.local.example .env.local
alembic -c alembic.ini upgrade head
python scripts/diagnose_auth.py --ensure-admin --password demo-pass
./start.sh
```

```bash
cd frontend
npm ci
npm run dev -- --host 0.0.0.0
```

Open `http://127.0.0.1:5173/idp/` and log in with `idp-admin / demo-pass`.

## Notes

- Real document parsing requires `MINERU_TOKEN` and a file URL reachable by MinerU cloud.
- Real AI extraction requires an OpenAI-compatible LLM key.
- Without provider keys, the system should still start, log in, and browse basic pages.
