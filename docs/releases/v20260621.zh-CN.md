# TechFlag IDP 社区版 v20260621

这个版本用于准备 GitHub 与 Gitee 的公开社区版发布。

## 重点更新

- 本地优先的智能文档处理工作台。
- 默认社区版配置，使用 SQLite 和本地对象存储。
- 支持配置 MinerU 文档解析 provider。
- 支持接入自己的 OpenAI-compatible LLM 进行 AI 抽取。
- 支持带证据的结构化抽取结果展示。
- 前端支持中文和英文。
- 补充公开 README、使用手册和安全产品截图。

## 启动方式

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -r requirements.txt
cp .env.local.example .env.local
alembic -c alembic.ini upgrade head
python scripts/diagnose_auth.py --ensure-admin --password demo-pass
./start.sh
```

```bash
cd frontend
npm ci
npm run dev -- --host 0.0.0.0
```

浏览器打开 `http://127.0.0.1:5173/idp/`，使用 `idp-admin / demo-pass` 登录。

## 说明

- 真实文档解析需要配置 `MINERU_TOKEN`，并确保 MinerU 云端可以访问上传文件 URL。
- 真实 AI 抽取需要配置 OpenAI-compatible LLM Key。
- 不配置 provider key 时，系统仍应能启动、登录并浏览基础页面。
